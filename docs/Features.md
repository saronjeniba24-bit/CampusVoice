# Features

- **User Authentication**
  - Signup and login for students and faculty
- **Complaint Submission**
  - Students can submit complaints with title, description, and optional attachments
- **Complaint Tracking**
  - Status updates: Pending, In Progress, Resolved
- **Dashboards**
  - Student Dashboard: My complaints
  - Faculty Dashboard: Complaints assigned to faculty
  - HOD Dashboard: Department-level complaints
- **Public Board**
  - Transparent view of all complaints for accountability
- **Timeline Page**
  - Chronological view of complaint updates and actions
