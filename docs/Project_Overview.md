# CampusVoice Project Overview

CampusVoice is a full-stack complaint management system designed for universities.  
It enables students to submit complaints, faculty/HODs to review them, and everyone to track resolution status transparently.

## Goals
- Provide a structured platform for complaint handling
- Ensure transparency and accountability
- Enable quick resolution through dashboards
- Store data securely in AWS DynamoDB
- Support multiple user roles (Student, Faculty, HOD)

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: AWS Lambda + API Gateway
- **Database**: DynamoDB
- **Hosting**: Netlify / AWS Amplify
